London Bike Sharing Analysis

Project Overview:

This project explores the patterns and drivers behind bike-sharing activity in London using data visualisation and exploratory analysis techniques. Insights are presented through a dynamic Tableau dashboard, which highlights weather conditions, user behaviors, and temporal trends.

Technologies Used:

  • Python (Pandas, Seaborn, Matplotlib)
  • Tableau (Dashboard Visualisation)
  • Jupyter Notebook

Key Features:

  • Preprocessed and cleaned raw usage data using Pandas
  • Created visualisations to explore seasonality, holidays, temperature, and humidity correlations
  • Published a public dashboard in Tableau for interactive exploration
  • Used Python for EDA, data wrangling, and correlation heatmaps
